## Tutorial hoe je de deckbuilder installeerd en gebruikt
<img width="1919" height="1079" alt="uitleg over mijn learningstory" src="https://github.com/user-attachments/assets/00c493aa-850b-475e-8a7e-f177c621c350" />

benodigdheden:
- nodejs
- express

als je het nog niet hebt installeer nodejs [hier](https://nodejs.org/en/download)

clone het project `git clone https://github.com/boi-one/deckbuilder`

ga naar de folder 

<img width="763" height="355" alt="image" src="https://github.com/user-attachments/assets/d8f7839c-b904-4aa4-a014-79185c99a7d4" />

<img width="769" height="414" alt="image" src="https://github.com/user-attachments/assets/f95e1fb2-d807-4bb1-ab75-70bfd854d39b" />

installeer express door `npm install express` te typen

typ daarna `node server.js`

<img width="332" height="164" alt="image" src="https://github.com/user-attachments/assets/38bcc54d-353e-4888-ba68-e3f84d64017c" />
